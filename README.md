# V2T (VPN TO TETHER) FOR ANDROID

## Membutuhkan 
- ROOT
- Iptables
- Terminal

## Cara Install
```
wget https://github.com/De4ce/v2t/raw/master/v2t
mv v2t /system/xbin/v2t
chmod 0755 /system/xbin/v2t
```
## Cara Menggunakan
```
su
v2t
```
## Antarnuka Default
- Bluetooth : bt-pan
- Wifi : wlan0
- USB : usb0
